# MindMelt Beta 2.0 – Feature Complete Static Beta

Flat deploy-safe package. Upload all files directly to the repository root. No build command, no npm, no containing asset folders.

## Added in Beta 2.0
- Multi-entry comma capture: separate thoughts with commas and MindMelt routes each one independently.
- Smarter universal capture routing for calendar, tasks, ideas, recommendations, notes, and brain dump.
- Smaller dashboard/category icons and cleaner widget area.
- Expanded Reframe ticker message library.
- Calendar Connections settings hub for Google, Apple Calendar, and Outlook planning.
- Static beta .ics calendar import for external calendar events.
- More robust notification settings with reminder presets, task/calendar toggles, and browser reminder support.
- Retains profile/settings, date setter, time setter, calendar recall, full-month calendar, productivity log, import/export, PWA icon, and deploy-safe static hosting support.
